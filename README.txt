FullStoreBA — Sitio listo para subir
Contenido: index.html + carpeta images/

Instrucciones rápidas para publicar:

1) Ver en tu PC
   - Descomprimí el ZIP y abrí index.html con doble clic en tu navegador.

2) Publicar en GitHub Pages (gratis)
   - Creá un repositorio en GitHub (público).
   - Subí los contenidos del ZIP a la rama `main` (o `gh-pages`).
   - En Settings -> Pages, elegí la rama `main` y la carpeta `/` como fuente.
   - En minutos tendrás: https://tuusuario.github.io/tu-repo/

3) Publicar usando Netlify Drop (sin cuenta opcional)
   - Accedé a https://app.netlify.com/drop y arrastrá el contenido del ZIP.
   - Netlify te dará una URL pública automática.

4) Formspree (formulario de contacto)
   - Para que los mensajes lleguen a fullstorebuenosaires@gmail.com entrá a https://formspree.io/ y creá un formulario gratuito.
   - Te darán un ID (por ejemplo `mxyzabq`). Reemplazá en index.html la acción del formulario:
     action="https://formspree.io/f/your-form-id" -> action="https://formspree.io/f/TU_FORM_ID"

5) Editar precios y textos
   - Abrí index.html y editá directamente en el navegador los textos (son contenteditable).
   - Para guardar cambios permanentes, abrí el archivo en un editor (VSCode/Notepad++) y pegá los textos en el HTML, o exportá el JSON con "Exportar JSON" y usalo para una tienda.
